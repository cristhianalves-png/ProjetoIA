Sobre o Projeto

O BookScript AI é um sistema de recomendação de livros desenvolvido utilizando conceitos de Inteligência Artificial, Machine Learning e Algoritmos de Busca. O objetivo é auxiliar leitores a encontrarem livros compatíveis com seus gostos de forma rápida e eficiente.

Como Funciona

1️⃣ Escolha um Gênero

Selecione a categoria de livros desejada.

2️⃣ IA Analisa os Dados

O algoritmo processa informações do catálogo.

3️⃣ Receba Recomendações

O sistema apresenta os livros mais relevantes.
